# Triển Khai API Serverless

### Tổng Quan
**Serverless** trên AWS cho phép bạn xây dựng và vận hành các ứng dụng mà không cần quản lý máy chủ. Dịch vụ như **AWS Lambda** tự động mở rộng quy mô và chỉ tính phí khi mã của bạn đang chạy, tối ưu hóa chi phí và tăng tính linh hoạt. Trong bài thực hành này, bạn sẽ học cách triển khai API serverless.
